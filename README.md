# GenFiles MCP (전북특별자치도청 캡스톤 프로젝트 커스터마이징 버전)

본 저장소는 오픈소스 GenFiles MCP를 기반으로 전북특별자치도청 캡스톤 프로젝트에서 사용하기 위해 커스터마이징한 버전입니다.  
Open-WebUI와의 연동을 중심으로 문서 생성 기능을 강화하고, 한글(HWP/HW/HWPX) 파일 및 품질 향상된 PPT 생성 기능을 추가하여 실사용 환경에 적합하도록 확장했습니다.  
민감한 정보는 포함하지 않으며, 프로젝트 범위 내에서 수정된 사항을 간단히 기술합니다.

---

## 주요 커스터마이징 내용

### 1. Open-WebUI 연동 기능 통합
- GenFiles MCP를 Open-WebUI와 직접 연동하도록 구조 개선
- WebUI에서 문서 생성 요청 시 곧바로 MCP를 통해 파일 생성 가능
- Open-WebUI 내 워크플로우와 자연스럽게 연결되도록 API 구조 정비

---

### 2. 한글(HWP/HW/HWPX) 문서 생성 기능 추가
- 오픈소스 MCP에는 없던 한글문서(HWP/HW/HWPX) 생성 기능 직접 구현
- 내부 행정 문서 포맷과 호환되도록 템플릿 구조 정리
- 한국어 기반 문서 생성 품질 개선

---

### 3. PPT 생성 기능 고품질화
- 기존 오픈소스 PPT 생성 기능을 전면 수정
- 슬라이드 구성, 텍스트 배치, 시각적 요소 품질을 개선하여 실제 보고 수준의 품질 확보
- 다양한 템플릿 구조 지원

---

### 4. 코드 및 운영 환경 개선
- 모듈 구조 정리 및 유지보수 용이성 강화
- 환경 변수를 통한 외부 서비스 연동 구조 개선
- 서버 운영 안정성 향상 및 예외 처리 강화
